## [0.3.0-alpha] - 2026-01-19

- 아키텍처: 멀티 레이어 시스템 도입 (Layer Composition)
- 기능: 레이어 추가/삭제/숨김/잠금 관리 기능
- UI: 레이어 관리 모달 및 툴바 연동
- 안정성: 레이어 잠금 시 편집 방지 로직 (Store/UI 이중 검증)

## [0.2.0] - 2026-01-19

- 입력/표시: 한글/이모지 등 전각·광폭 문자 셀 폭(2칸) 처리 및 커서 이동 정합
- 최적화: 클립보드 붙여넣기 성능 개선 (Batch Processing 도입으로 시간 복잡도 대폭 감소)
- 편집: 선택 도구에서 클릭 위치 기준으로 붙여넣기(바운더리 밖은 자동 절단)
- 단축키: Ctrl/Cmd+C/X/V, Ctrl/Cmd+Z/Y, Ctrl/Cmd+Shift+Z, Delete 지원 정리
- UI: 편집기 반투명 격자(셀 크기) 및 버퍼 경계선(작업 영역) 표시
- 입력: Enter 시 클릭 기준 열 유지(바로 아래 칸으로 이동)
- UI: 선택 영역 우클릭 컨텍스트 메뉴(복사/잘라내기) 및 빈 영역 우클릭 붙여넣기
- 편집: 찾기/바꾸기 매치 하이라이트 및 이전/다음 탐색
- 빌드: 배포 산출물(exe/msi/nsis)을 `out/` 폴더로 자동 복사, `build:exe` 명령 추가
- 빌드: macOS(dmg) 및 Linux(deb/appimage) 크로스 플랫폼 빌드 산출물 자동 처리 지원
- UI/편집: 최근 파일/고정 파일 관리 및 깨진 경로 자동 정리
- UI: 리본 메뉴(도구/옵션) 추가
- 편집: 사각형 블록 선택 및 복사/잘라내기/붙여넣기/삭제
- 드로잉: 사각형/직각선/화살표/프리폼(#/$/%) 텍스트 드로잉
- 드로잉: ASCII/유니코드 문자 스타일 선택

## [0.1.0] - 2026-01-17

- 구조: 100% 텍스트 버퍼 기반 편집으로 전환
- 버퍼: 템플릿(80×24/120×80/160×100) 및 커스텀 크기 생성
- UI: 텍스트 버퍼 편집 화면 및 Ctrl/Cmd+S 저장
- 파일: 텍스트 파일 열기(.txt)
- 설정: 언어/개행(LF/CRLF)/저장 시 우측 공백 채우기/마지막 파일 경로 영속화
- 편집: Undo/Redo(Ctrl/Cmd+Z, Ctrl/Cmd+Shift+Z, Ctrl/Cmd+Y)
- Backend: 텍스트 파일 저장 커맨드 유지(write_text_file)
- Backend: 텍스트 파일 읽기 커맨드 추가(read_text_file)
- 정리: Konva/도형 모델 및 HTML/PNG Export 코드 제거
